# Paper folder template

Copy this folder when starting a new paper idea.

Recommended files:

```text
paper-folder/
├── README.md
├── paper.yaml
├── concept-brief.md
├── venue-strategy.md
├── knowledge-base.md
├── outline.md
└── notes/
    └── README.md
```

## Minimum contents

- Working title
- One-sentence paper idea
- Research questions
- Main contribution
- Target venue
- Backup venues
- Required knowledge
- Empirical material needed
- Next actions
